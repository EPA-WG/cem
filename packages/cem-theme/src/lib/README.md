# CEM Theme Library

This is the core theme library for the **CEM** component system.

## Features

- Modern ES2022 output
- TypeScript definitions
- XHTML documentation

## Usage

```typescript
import { cemTheme } from '@epa-wg/cem-theme';

const theme = cemTheme();
console.log(theme);
```
